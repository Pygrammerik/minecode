Переменные

Основные типы переменных для хранения данных в скриптах MineCode

int - Целое число

Числовой тип
int "имя" = "число"

Переменная содержащая целое число

Примеры:

int "health" = "20" int "blocks" = "64" int "count" = "0"

string - Строка

Текстовый тип
string "имя" = "текст"

Переменная содержащая текст

Примеры:

string "message" = "Привет, мир!" string "playerName" = "Steve" string "status" = "Работает"

bool - Логическое значение

Булевый тип
bool "имя" = "true/false"

Переменная содержащая true или false

Примеры:

bool "isAlive" = "true" bool "hasArmor" = "false" bool "isRunning" = "true"

Block - Блок

Блочный тип
Block "имя" = "id блока"

Переменная содержащая id блока

Примеры:

Block "stone" = "1" Block "diamond" = "57" Block "wood" = "17"

Действия

Основные команды для выполнения действий в игре

say - Отправка сообщения

Чат
say "сообщение"

Выводит сообщение в чат игроку

Примеры:

say "Привет, мир!" say "Скрипт выполнен!" say "Добыто блоков: " + blocks

wait - Ожидание

Время
wait "число"

Ожидание (в десятых секунды)

Примеры:

wait "10" // 1 секунда wait "50" // 5 секунд wait "100" // 10 секунд

jump - Прыжок

Движение
jump

Прыжок игрока

Пример:

jump wait "5" jump

follow - Слежка

AI
follow "число"

Слежка за существом на которое смотрел (в секундах)

Примеры:

follow "30" // Следить 30 секунд follow "60" // Следить 1 минуту

Движение

Команды для управления движением игрока

run - Ходьба

Движение
run "число"

Пройти указанное количество блоков вперед

Примеры:

run "10" // Пройти 10 блоков run "1" // Пройти 1 блок run "50" // Пройти 50 блоков

run.sprint - Бег

Быстрое движение
run.sprint "число"

Пробежать указанное количество блоков

Примеры:

run.sprint "20" // Пробежать 20 блоков run.sprint "5" // Пробежать 5 блоков

run.right - Движение вправо

Боковое движение
run.right "число"

Пройти вправо указанное количество блоков

run.left - Движение влево

Боковое движение
run.left "число"

Пройти влево указанное количество блоков

run.back - Движение назад

Обратное движение
run.back "число"

Пройти назад указанное количество блоков

Взаимодействие

Команды для взаимодействия с миром

clickLeft - Левый клик

Мышь
clickLeft

Имитирует левый клик мыши

clickRight - Правый клик

Мышь
clickRight

Имитирует правый клик мыши

breakBlock - Разрушение блока

Добыча
breakBlock

Разрушает блок на который смотрит игрок

Управление камерой

Команды для управления камерой игрока

cameraRotation - Поворот камеры

Камера
cameraRotation "число" "число"

Добавляет к текущему положению камеры значения

Примеры:

cameraRotation "90" "0" // Повернуть на 90° вправо cameraRotation "0" "45" // Поднять камеру на 45°

setCameraRotation - Установка камеры

Камера
setCameraRotation "число" "число"

Устанавливает камеру в определённое направление

Примеры:

setCameraRotation "180" "0" // Смотреть назад setCameraRotation "0" "90" // Смотреть вверх

Информация об игроке

Переменные для получения информации об игроке

player.underBlock - Блок под игроком

Информация
player.underBlock

Возвращает блок на котором стоит игрок

player.targetBlock - Целевой блок

Информация
player.targetBlock

Возвращает блок на который смотрит игрок

player.health - Здоровье

Здоровье
player.health

Возвращает текущее здоровье игрока

Пример:

if (player.health < 10) say "Мало здоровья!"

Логические конструкции

Конструкции для создания условной логики и циклов

if - Условный блок

Условие
if (условие)

Начинает условный блок

Пример:

if (player.health < 10) say "Мало здоровья!" wait "10"

else - Альтернативный блок

Условие
else

Альтернативный блок после if

Пример:

if (player.health < 10) say "Мало здоровья!" else say "Здоровье в порядке!"

for - Цикл

Цикл
for (int; условие; действие)

Создает цикл

Пример:

for (int i = 0; i < 5; i = i + 1) say "Шаг " + i wait "10"
